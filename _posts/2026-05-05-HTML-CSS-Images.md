Salut les codeurs du dimanche *(et les pros aussi, on ne juge pas)* !  
  
Aujourd'hui, on parle **rangement**. Oui, oui, comme dans votre chambre — mais version **code**. On va dompter le chaos des fichiers HTML, CSS et images pour un site web qui respire la *zen attitude*.  
  
---  
  
## 🧱 HTML : La Carcasse *(Sans les Os qui Dépassent)*  
  
Le **HTML**, c'est le squelette de votre site.  
Imaginez-le comme un meuble IKEA : des balises partout, mais une fois assemblé, ça tient debout.  
  
👉 Utilisez les balises sémantiques comme :  
- `<nav>`  
- `<article>`  
- `<header>`  
- `<footer>`  
  
...pour que Google comprenne de quoi vous parlez.  
  
---  
  
## 🎨 CSS : La Touche de Glamour *(Sans Virer Paillettes)*  
  
Le **CSS**, c'est le maquillage de votre site :  
couleurs, polices, effets spéciaux…  
  
⚠️ Attention à ne pas en abuser, sinon votre site ressemblera à un sapin de Noël en plein mois d'août.  
  
🎯 Bonnes pratiques :  
- Utilisez des **classes** (`.classe`)  
- Utilisez des **ID** (`#id`)  
- Gardez un style **cohérent et lisible**  
  
---  
  
## 🖼️ Images : Les Stars du Show *(Mais Pas Trop Gourmandes)*  
  
Les images, c'est le casting de votre site.  
  
Formats courants :  
- **JPEG** → photos  
- **PNG** → transparence  
- **SVG** → logos & icônes  
  
🚀 Astuce :  
Optimisez-les ! Personne n'aime un site qui met 3 plombes à charger *(sauf votre concurrent, peut-être)*.  
  
---  
  
## ✅ Conclusion : Le Bonheur est dans le Dossier Rangé  
  
Alors, on récapitule :  
- 📄 Un fichier **HTML** pour la structure  
- 🎨 Un fichier **CSS** pour le style  
- 📁 Un dossier bien organisé pour les images  
  
C'est comme ranger vos chaussettes par couleur :  
ça ne change pas le monde, mais ça fait du bien.
