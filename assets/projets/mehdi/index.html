Mes objectifs personnels

  • Apprendre HTML

    • Apprendre le HTML est la base de la création de sites web. Cela permet de structurer une page en utilisant des balises comme les titres, les paragraphes, les images ou les liens. Grâce à ce langage, on apprend à organiser correctement le contenu d’un site pour qu’il soit lisible et compréhensible par un navigateur. C’est une étape essentielle pour comprendre comment fonctionne le web, car chaque site internet repose sur du HTML. Cela permet aussi de poser les bases pour apprendre d’autres langages par la suite.

  • Comprendre CSS

    • Le CSS sert à donner du style à une page web. Il permet de modifier les couleurs, les polices, les tailles ou encore la disposition des éléments. Comprendre le CSS est essentiel pour rendre un site plus agréable visuellement et créer une véritable identité graphique. Grâce à lui, une simple page HTML peut devenir un site moderne et attractif. Il permet aussi de mieux organiser la présentation des éléments.

  • Créer un site propre

    • Créer un site propre signifie organiser correctement son code et son contenu. Cela passe par une structure claire, une bonne hiérarchie des informations et un design cohérent. Un site propre est plus facile à lire, à modifier et plus agréable pour les utilisateurs. Il est aussi plus professionnel et plus simple à améliorer dans le temps. C’est une bonne pratique importante en développement web.

  • Réussir ma formation

    • Réussir ma formation PSIN est un objectif important qui demande du travail et de la régularité. Cela implique d’apprendre les bases du développement web, de pratiquer souvent et de progresser étape par étape afin d’acquérir de bonnes compétences en informatique. Il faut aussi être organisé et persévérant pour avancer correctement. L’objectif est de maîtriser les outils numériques et de se préparer à un futur professionnel dans ce domaine.
