Le groupe Astrolabe a travaillé sur un projet complet en HTML et CSS : créer une page web intitulée **Mon parcours de formation PSIN**.

L'objectif était de réutiliser les bases vues en formation : structure HTML, mise en forme CSS, images, liens, listes et organisation des contenus.

## Consigne courte

Créer un mini site avec :

- un fichier `index.html` ;
- un fichier `styles.css` ;
- au moins une image ;
- une présentation de l'entrée en formation ;
- un calendrier avec les étapes importantes ;
- une liste d'objectifs personnels ;
- des liens vers Google, Wikipédia et une page personnelle.

La page devait utiliser les principales balises HTML vues ensemble : titres, paragraphes, liens, images, listes, sections et classes CSS.

Le CSS devait contenir les propriétés de base : couleurs, police, taille du texte, bordures, arrondis, alignement, marges et espacements.
