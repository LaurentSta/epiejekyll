Le groupe Gidef travaille sur une histoire interactive en HTML : un récit dans lequel le lecteur avance en cliquant sur des liens et choisit son propre chemin.

L'objectif est de réutiliser les bases HTML vues en séance pour créer une expérience simple, lisible et fonctionnelle.

![Banderole du projet histoire interactive]({{ '/assets/images/gidef/HistoireBanderole.png' | relative_url }})

## Consigne courte

Créer une histoire de type **dont vous êtes le héros** avec :

- 20 pages HTML ;
- une première page appelée `index.html` ;
- un titre sur chaque page ;
- un texte sur chaque page ;
- 2 ou 3 liens vers d'autres pages ;
- plusieurs chemins possibles ;
- plusieurs fins possibles ;
- un code HTML propre et bien organisé.

Le lecteur doit pouvoir avancer dans l'histoire en cliquant sur les liens, comme dans un livre interactif.

## Rappel HTML

Les stagiaires peuvent utiliser les balises vues en séance :

- `<!DOCTYPE html>`
- `<html lang="fr">`
- `<head>` et `<body>`
- `<meta charset="UTF-8">`
- `<title>`
- `<h1>`, `<h2>`, `<h3>`
- `<p>`, `<strong>`, `<em>`, `<br>`
- `<ul>`, `<ol>`, `<li>`
- `<a href="page1.html">Page 1</a>`
