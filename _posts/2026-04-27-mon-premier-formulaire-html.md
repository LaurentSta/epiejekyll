Le groupe Astrolabe travaille sur un nouveau projet : créer un formulaire HTML capable de récupérer les réponses d'un utilisateur et de générer un fichier CSV.

L'objectif est de réutiliser les bases du HTML, puis d'ajouter un peu de JavaScript pour rendre le formulaire fonctionnel.

## Consigne courte

Créer une page `index.html` contenant :

- un titre ;
- un texte d'explication ;
- un formulaire fonctionnel ;
- un champ texte pour le nom ou le prénom ;
- une question à choix unique avec des boutons radio ;
- une question à choix multiple avec des cases à cocher ;
- une liste déroulante ;
- une zone de commentaire ;
- un bouton d'envoi.

Le formulaire doit utiliser JavaScript pour récupérer les réponses, créer un fichier CSV et déclencher son téléchargement.

Le thème est libre : habitudes numériques, loisirs, préférences, musique, jeux, films ou utilisation d'Internet.

## Niveaux possibles

- Débutant : formulaire fonctionnel et CSV généré.
- Intermédiaire : ajouter une question, plusieurs choix et une meilleure mise en forme.
- Avancé : ajouter une validation, un message de remerciement et stocker plusieurs réponses avec `localStorage`.
