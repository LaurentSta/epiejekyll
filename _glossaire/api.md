Une **API** (Application Programming Interface) est un contrat technique.

Elle definit quelles donnees peuvent etre demandees, comment les demander, et sous quel format la reponse est renvoyee.

Dans un projet web, une API peut par exemple connecter un site a un service externe (cartes, paiement, statistiques, etc.).
